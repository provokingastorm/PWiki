
## Description
As a user, I want to be able to input the input sequence that will be exported to a cronus script by text, where the input is separated by commas. 

## Acceptance Criteria
 - The program provides a button to append the current input sequence to running input command list. 
 - The text box will only accept input for the Xbox controller. (PC and Playstation input will be supported in the future).
 - The user is prevented from entering a comma without an input sequence between.
 - The input will be display in upper-case letters.
 - The programs displays an error message for any input that cannot be exported when the user selected the "Add Input" button.
